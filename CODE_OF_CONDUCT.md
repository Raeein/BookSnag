# Code of Conduct

BookSnag follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good intent, and focus on the work.

## Enforcement

Report unacceptable behavior by opening a GitHub issue. Maintainers will review and respond as appropriate, up to and including removing comments, commits, or contributors from the project.
